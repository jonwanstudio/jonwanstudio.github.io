Emergency Awareness Device (EAD)

ML-Powered Alerts for Auditory Disabilities

About This Project

This project is a real-time device that integrates sensor technology and machine learning to immediately detect alarm signals, such as fire alarms or smoke detectors.

The primary objective is to enhance safety and reduce the gap in emergency alarm systems for people with auditory disabilities by alerting them through powerful vibrations and clear visual notifications on a built-in screen.

Outcomes & Contributions

  • Created a real-time device to detect fire alarm sounds.
  • Designed the machine learning algorithm with reinforcement learning for audio recognition and implemented it onto an Arduino board.
  • Trained the machine learning algorithm with a variety of fire alarm samples found online, achieving 98% accuracy in testing.
  • Placed 2nd in the FAS Speak Out Competition 2024 out of six finalist teams.

Key Features

  • Real-Time Detection: Instantly identifies acoustic alarm signals.
  • Machine Learning: Uses an ML model trained to recognize specific emergency sounds.
  • Haptic Alerts: Provides powerful vibrating feedback.
  • Visual Notifications: Displays clear, simple alerts on a dedicated screen.
  • Accessibility Focus: Designed to enhance safety for the hearing impaired.
  • Sensor Integration: Combines microphone and processing hardware in a compact unit.

Technical Skills

  • Additive Manufacturing
  • Machine Learning (ML)
  • Arduino Implementation
